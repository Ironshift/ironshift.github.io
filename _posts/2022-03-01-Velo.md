# Objectif

Rénovation complète d’un vélo de descente de montagne destiné à être électrifié : remise en état du cadre et des composants, choix et préparation de l’intégration du moteur. Dimensionnement de la batterie et de l’électronique associée.

## Choix Vélo

<!-- markdownlint-capture -->
<!-- markdownlint-disable -->
> Modèle retenu : `Rockrider 9.1`
{: .prompt-tip }
<!-- markdownlint-restore -->

![Desktop View](/assets/velo/rockrider9.1.webp)

Avantages :

- Prix abordable (300 € d'occasion en excellent état).
- Vélo tout-suspendu, offrant un bon confort et permettant de circuler facilement sur des sentiers escarpés, y compris en milieu urbain.
- Freins à disque hydrauliques, performants et indispensables pour maîtriser l’arrêt du vélo une fois électrifié et donc nettement plus lourd.

Inconvénients :

- Poids élevé : 13.9 Kg
- Format de roues de 26 pouces, les vélos actuels ont tendances à être de 27.5 voire 29 pouces --> augmentation de la stabilité et de la capacité à franchir les obstacles.

## Choix Motorisation

La question entre deux technologies de motorisation se pose, un moteur dans le pédalier ou dans la roue :


|                     | Avantages                       | Inconvénients                     |
| :------------------ | :------------------------------ | :-------------------------------- |
| **Moteur pédalier** | Bon équilibre du vélo           | Prix élevé                        |
|                     | Fort couple                     | Usure chaîne/cassette             |
|                     | Roues faciles à entretenir      | Montage plus complèxe             |
|                     | Aide plus naturelle             | Légère résistance sans assistance |
|                     |                                 |                                   |
| **Moteur roue**     | Prix bas                        | Moins bon en montée               |
|                     | Montage simple                  | Poids mal réparti                 |
|                     | Pas d’usure de la transmission  | Crevaison plus pénible            |
|                     |                                 | Moins naturel à l’usage           |
|                     |                                 | Tension sur les rayons            |
|                     |                                 | Effet gyroscopique                |

## Choix Batterie & Dimensionnement 

Le choix de la batterie a été limité par un problème d’encombrement. Sur un vélo sans suspension arrière, il est facile d’installer la batterie dans le triangle du cadre ; avec une suspension arrière, cela devient beaucoup plus difficile.
La solution retenue est donc une batterie qui s'extrait par le côté plutôt que par le haut, comme c’est le cas pour la plupart des batteries destinées aux kits de conversion électrique.
